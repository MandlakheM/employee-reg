## EMPLOYEE REGISTRATION WEB-APP

**SUMMARY**

This is an employee registration app thats mainly created for admins to be able to be able to: 

i) register new employees,

ii) view registerd employees,

iii) update employees information 

iv) and remove employees from the current ones. deleted employees will be able to be viewed in a different page.
